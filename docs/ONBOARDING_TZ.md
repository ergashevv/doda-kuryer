# Техническое задание: онбординг (BK-бот)

Версия: финальная по состоянию на 2026-04-03. Реализация: `src/bk/handlers.js`, `src/bk/yandexFlow.js`, `src/flow.js` (Doda).

## Общая последовательность

1. **Выбор языка** (uz / ru / tg / ky).
2. **Номер телефона (РФ)** — сразу после языка, до основного меню.
3. **Главное меню** (в парке / не в парке / поддержка).
4. Если **«не в парке»** и телефон уже есть → **выбор сервиса**:
   - RU-парк Doda, курьер
   - Яндекс Лавка
   - Яндекс Еда

**Яндекс Лавка** и **Яндекс Еда** — одинаковые шаги; в данных сохраняется `service`: `yandex_lavka` | `yandex_eda`.

## RU-парк Doda, курьер (существующая логика, не сокращать)

- Категория транспорта → (легковой: СТС РФ или 2× техпаспорт) → город → вопрос «гражданин РФ?» → цепочка `dodaDocSequence` из `src/flow.js` → проверка → отправка.

Подробная таблица ключей: см. комментарии в `src/flow.js` и раздел в исходном плане онбординга.

## Яндекс Лавка / Еда

1. Город: **Москва** или **Санкт-Петербург**.
2. Гражданство: **Узбекистан/Таджикистан** | **Казахстан/Кыргызстан** | **РФ** | **Туркменистан**.

### Узбекистан / Таджикистан

Тип документа: **Патент** | **Вид на жительство** | **Студенческий**.

- **Патент:** паспорт (разворот с фото) → патент лицевая/оборот → регистрация (2 фото) или Амина (1 скрин) → миграционная карта → оплата патента (фото + файл) → реквизиты (видео, фото, текст) → 16 цифр карты → телефон привязки карты.
- **ВНЖ:** ВНЖ лицевая и оборотная → миграционная карта → далее как у гражданина РФ по финансовому блоку: телефон для связи → реквизиты (видео, фото, текст) → 16 цифр → телефон привязки.
- **Студенческий:** студенческий билет → справка об обучении → паспорт (разворот) → регистрация или Амина → миграционная карта → реквизиты → 16 цифр → телефон привязки.

### Казахстан / Кыргызстан

Паспорт или ID → (паспорт: разворот с фото | ID: лицевая и оборотная) → миграционная карта → прописка/регистрация или Амина → реквизиты → 16 цифр → телефон привязки.

### РФ

Паспорт (страница с фото) → страница с пропиской → телефон для связи → реквизиты → 16 цифр → телефон привязки.

### Туркменистан

Паспорт (разворот с фото) → тип визы (рабочая / студенческая) → фото визы → регистрация или Амина → миграционная карта → телефон для связи → реквизиты → 16 цифр → телефон привязки.
